#My Udemy Clone Website

A simple Udemy clone website built using HTML and basic CSS.

## 📌 Project Description

This project is a basic clone of the Udemy website created using HTML and CSS. It includes essential sections such as a navigation bar, course categories, featured courses, and a footer. The purpose of this project is to practice front-end web development skills and improve proficiency in HTML and CSS.

## 🛠️ Technologies Used

- HTML5
- CSS3

## ✨ Features

- Responsive Navigation Bar
- Course Listing Section
- Categories Section
- Clean and Simple UI
- Footer Section

## 🚀 Purpose

This project was developed to enhance my understanding of HTML and CSS by recreating the layout of the Udemy website.
